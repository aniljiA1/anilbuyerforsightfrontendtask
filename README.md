# 🚀 User Dashboard

A modern **User Directory Dashboard** built using **React + Vite**, which displays users, supports search, sorting, and detailed user views.

## 📖 Project Overview

This project is part of a **Frontend Engineer Assessment**.

It fetches user data from a public API and provides:

* 📊 Dashboard view (table/grid)
* 🔍 Search functionality (Name & Email)
* 🔄 Sorting (Name & Company)
* 👤 User Detail Page

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🌐 Axios (API calls)
* 🔁 React Router DOM (Routing)
* 🎨 CSS (Custom styling)

---

## 🌐 API Used

https://jsonplaceholder.typicode.com/users

---

## ✨ Features

### ✅ Dashboard

* Displays users in a clean table
* Shows:

  * Name
  * Email
  * Phone
  * Company

### 🔍 Search

* Filter users by:

  * Name
  * Email
* Real-time client-side filtering

### 🔄 Sorting

* Sort by:

  * Name
  * Company
* Toggle:

  * Ascending (↑)
  * Descending (↓)

### 👤 User Detail Page

* Click on any user row
* Navigate to detailed user page
* Displays:

  * Full name
  * Email
  * Phone
  * Username
  * Website
  * Company details
  * Address
